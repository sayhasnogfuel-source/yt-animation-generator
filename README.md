# YouTube Shorts Animation Generator (Remotion + React + TypeScript)

This project now starts with one polished sample video: **“He wouldn't share his chips”**.

## First preview target

- Composition ID: `BlockyChipsPreview`
- Style: original kid-friendly blocky 3D-style family skit
- Format: 1080x1920, 30 FPS, top subtitle bar, fast reaction cuts

## Exact commands

### Preview in Remotion Studio

```bash
npm run studio
```

### Render the polished sample MP4 first

```bash
npm run render:chips
```

### Output path

```txt
out/chips-preview.mp4
```

## After previewing this sample

You can still render the reusable template version:

```bash
npm run render:blocky
```

Output:

```txt
out/blocky-family-short.mp4
```
